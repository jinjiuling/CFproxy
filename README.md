# CFproxy

* 详细使用转[博客](https://bk.shunleite.com/post-51.html)

* 如果出现谷歌搜索出现流量异常的情况请在谷歌搜索的网址后面添加.hk或者.jp再或者.jk等试着解决（一般.hk）
  
* 或者你直接在首页栏目里直接输入你想搜索的东西然后点击“开始通向”即可（两者总有一个可以​）

  例如：https://dl.slt6.com/-----https://www.google.com.hk
